## Современные технологии программирования

Репозиторий содержит решенные задания по курсу "Современные технологии программирования", реализованные на языке Java.

**Внимание:** Для запуска **Сложных вариантов** необходимо наличие запущенного локального сервера **MySQL**. Подключение осуществляется интерактивно при старте программы (запрашивается имя БД, логин и пароль). В ходе первого сохранения данных в Excel может появиться техническое предупреждение (`ERROR`) от Apache POI, связанное с отсутствием логирования; на функциональность это не влияет.

---

## Структура проекта (Архитектура)

Проект использует **модульную** и **MVC-подобную** структуру для разделения ответственности:
- `common/` - Общие компоненты, используемые всеми заданиями (Контроллер, Базовая модель, Класс для сохраненных запросов).
- `pr1/` - Задание 1: Математические операции с сохранением в MySQL и экспортом в Excel.
- `pr2/` - Задание 2: Манипуляции с двумя строками.
- `pr3/` - Задание 3: Валидация числовых данных.
- `pr4/` - Задание 4: Методы работы со строками (`substring`, регистры).

---

## Задание 1: Арифметические операции (MySQL + Excel)

**Сложный вариант.** Программа реализует интерактивное консольное меню для выполнения математических операций (сложение, вычитание, умножение, деление, остаток, модуль, степень) над целочисленным, байтовым и вещественным типами данных.

* **Особенность:** Результаты всех операций **сохраняются в MySQL**, а пункт 10 позволяет экспортировать все накопленные данные в файл **Excel (`.xlsx`)**.

| Файл | Роль |
| :--- | :--- |
| `src/main/java/pr1/Calculator.java` | Точка входа (Main). |
| `src/main/java/pr1/model/Arithmetic.java` | **Модель** (логика вычислений и меню). |

## Общие Компоненты (Пакет `common`)

Ключевые классы для обеспечения работы всей архитектуры:
- **`Control.java`**: Контроллер. Управляет циклом команд, **устанавливает соединение с MySQL** и обрабатывает ошибки.
- **`Model.java`**: Абстрактная модель. Содержит базовый функционал для работы с **БД** (создание таблиц, экспорт в Excel), а также **внутренний класс `IO`** для унифицированного ввода-вывода через `Scanner`.
- **`Save.java`**: Класс-запись для структурированного хранения данных о результатах, сохраненных в БД.
